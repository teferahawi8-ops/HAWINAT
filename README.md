# HAWINAT 🎬

## About The Project

HAWINAT is a personal educational project built with **React** and **Node.js** to practice modern web development and API integration.

The application allows users to:
- Browse movies
- Search for movie titles
- View detailed movie information
- Rate movies
- Save favorite movies

Movie data is provided through the **TMDB API**.

This project was created for learning purposes and to demonstrate web development skills as part of my portfolio.

---

## Technologies Used

- React.js
- Node.js
- TMDB API
- JavaScript
- HTML
- CSS

---

## Purpose

The main goal of HAWINAT is to improve my understanding of:
- Frontend development with React
- Backend development with Node.js
- Working with external APIs
- Building interactive web applications
